# 📖 What is Python?

Python is a **high-level, interpreted, and general-purpose programming language** created by **Guido van Rossum** and first released in **1991**.

It is designed to be easy to read, write, and understand, making it one of the most popular programming languages in the world.

Python is widely used for:

- 🌐 Web Development
- 🤖 Artificial Intelligence (AI)
- 📊 Data Science
- 📈 Data Analysis
- ☁️ Cloud Computing
- 🖥️ Automation & Scripting
- 🎮 Game Development
- 📱 Desktop Applications
- 🔒 Cyber Security
- 🧠 Machine Learning

# 🛠 Install Python

## Step 1

Download Python from the official website.

👉 https://www.python.org/downloads/

## Step 2

Install Python by following the setup wizard.

✔️ During installation, make sure to select:

```
☑ Add Python to PATH
```

## Step 3

Verify the installation.

Open **Command Prompt** or **Terminal** and run:

```bash
python --version
```

or

```bash
python3 --version
```

Example output:

```text
Python 3.13.5
```
