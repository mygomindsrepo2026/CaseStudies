# ➕ Operators in Python

# 📖 Theory

An **operator** is a special symbol that tells Python to perform a specific operation on one or more operands (values or variables).

For example,

```python
a = 10
b = 20

print(a + b)
```

Output

```
30
```

Here,

- `a` and `b` are **operands**
- `+` is the **operator**

---

# ❓ Why Do We Need Operators?

Operators help us:

- Perform mathematical calculations
- Compare values
- Make decisions
- Combine conditions
- Assign values
- Check object identity
- Check membership in collections
- Perform bit-level operations

Without operators, programming would be nearly impossible.

- Work with all built-in data types

# ➕ Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

| Operator | Name | Example |
|----------|------|---------|
| + | Addition | a + b |
| - | Subtraction | a - b |
| * | Multiplication | a * b |
| / | Division | a / b |
| // | Floor Division | a // b |
| % | Modulus | a % b |
| ** | Exponent | a ** b |

# 🟰 Assignment Operators

Assignment operators are used to assign values to variables.

| Operator | Example | Equivalent To |
|----------|---------|---------------|
| = | x = 10 | Assign value |
| += | x += 5 | x = x + 5 |
| -= | x -= 5 | x = x - 5 |
| *= | x *= 5 | x = x * 5 |
| /= | x /= 5 | x = x / 5 |
| //= | x //= 5 | x = x // 5 |
| %= | x %= 5 | x = x % 5 |
| **= | x **= 2 | x = x ** 2 |

---

## 📝 Example 1 : Assignment (=)
